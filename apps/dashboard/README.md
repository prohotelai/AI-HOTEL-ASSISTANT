# Dashboard App

Next.js 15 App Router dashboard scaffold for AI Hotel Assistant V2.

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run start` — start built app
- `npm run lint` — run Next.js lint

## Notes

TailwindCSS and Shadcn UI-ready via shared UI package under `packages/ui`. Add routes and components under `app/`.
