# AI-HOTEL-ASSISTANT
A comprehensive multi-tenant SaaS platform offering AI-powered conversational agents, knowledge base management, website auto-scanning, and PMS integrations for hotels.

## Vercel Install / Configure

1. Install the Vercel CLI: `npm i -g vercel`
2. Authenticate: `vercel login`
3. From the repository root, link the project to Vercel: `vercel link`
4. Deploy (or create a preview): `vercel`

The linked project will store environment variables and build settings in Vercel. Use `vercel env pull` to sync local `.env` values once they are defined in the Vercel dashboard.
